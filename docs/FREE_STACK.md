# Free Stack

Use free/open-source first.

| Need | Free path | Notes |
|---|---|---|
| App framework | Next.js | included starter |
| UI | Tailwind | included starter |
| Tests | Vitest | included starter |
| Agent runtime | Claude Code | user installs |
| Local models | Ollama | optional fallback |
| Code intelligence | Serena MCP | optional install |
| Memory graph | Graphiti | optional install |
| Notes | Obsidian | optional desktop app |
| Obsidian automation | Local REST API plugin | optional key |
| Repo automation | GitHub CLI | optional login |
| Database | SQLite local, Supabase free | choose per project |
| Deploy | Vercel free, Render free, Fly free tier | watch limits |

Paid APIs can be added later, but the project must start useful without them.
