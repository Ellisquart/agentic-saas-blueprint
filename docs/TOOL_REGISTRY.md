# Tool Registry

Use this status language everywhere.

| Status | Meaning |
|---|---|
| Verified | installed, configured, tested |
| Configured | config exists, needs user key/app |
| Scaffold Only | file shape exists, not working |
| Blocked | user/vendor action required |

## Recommended Tools

| Tool | Default status | Purpose |
|---|---|---|
| Claude Code | Configured | main coding agent |
| Serena MCP | Scaffold Only | semantic code search/editing |
| Graphiti | Scaffold Only | temporal knowledge graph memory |
| Obsidian | Configured | local notes and decisions |
| Obsidian Local REST API | Scaffold Only | note read/write automation |
| Ollama | Scaffold Only | local model fallback |
| GitHub CLI | Scaffold Only | repo, issue, PR automation |

## Rule

Do not say a tool works until a command or UI check proves it.
